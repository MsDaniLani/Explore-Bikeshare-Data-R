# Function to view column names, number of columns, and number of rows.

view_data_info <- function(data) {
  #'''
#' This function takes a dataframe as input and prints the column names, number of columns, and number of rows.
#'
#' parameter - data - A dataframe.
#' return value - None - prints the information to the console
#'''
  cat("Column Names:\n")
  print(colnames(data))
  cat("\nNumber of Columns: ", ncol(data))
  cat("\nNumber of Rows: ", nrow(data), "\n")  
}

Summary of Question 1¶

The most common month is 06 which is June. The data used is based on northern states. During the month of June, the weather is generally warm and pleasant, which may contribute to the high frequency of bike trips.¶
The most common day of the week is Wednesday, suggesting that people may be using bikeshare for commuting or other regular activities during the workweek.¶
The most common hour of day is 08, which is likely a peak commuting time. This information can be useful for bike sharing companies to optimize their operations and ensure that there are enough bikes available during high-demand times.¶
